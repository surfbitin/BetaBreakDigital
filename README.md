# BetaBreakDigital

A personal progressive web app based on [Beta Break](https://www.betabreakgame.com/) (not my creation).

## Overview

**BetaBreakDigital** is a standalone HTML application inspired by the original Beta Break Game. It is designed to leverage Progressive Web App (PWA) features, allowing for flexible self-hosting or offline use.

## Features

- 100% HTML (no backend required)
- Progressive Web App ready
- Can be hosted on any static server or run locally
- Works in modern browsers

## Getting Started

### Installation

You can use BetaBreakDigital in two ways:

#### 1. Self-Hosted
- Clone/download the repository
- Place the HTML file(s) on your web server
- Access via your browser

#### 2. Local Usage
- Download the HTML file(s)
- Double-click or open in any browser

### PWA Features

- Add to Home Screen
- Offline support (when served with a manifest/service worker)
